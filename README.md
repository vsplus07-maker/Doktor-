# 🏥 Доктор - Остеопат

**Профессиональный сайт остеопата, блог и личная визитка**

Современный, отзывчивый лендинг для врача-остеопата с интегрированным блогом и информацией об услугах.

## ✨ Возможности

✅ **Мобильный дизайн** - Адаптивный интерфейс для всех устройств  
✅ **Профессиональный вид** - Современный и доверительный дизайн  
✅ **Галерея услуг** - Описание всех предоставляемых услуг  
✅ **Блог** - Полезные статьи о здоровье и остеопатии  
✅ **Контактная форма** - Запись на прием и обратная связь  
✅ **SEO готов** - Оптимизирован для поисковых систем  
✅ **Социальные сети** - Ссылки на все каналы  
✅ **Анимации** - Плавные эффекты и переходы

## 🚀 Установка и использование

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/vsplus07-maker/Doktor-.git
   cd Doktor-
   ```

2. **Откройте в браузере**
   - Просто откройте файл `index.html` в вашем браузере
   - Или используйте Live Server в VS Code

3. **Кастомизируйте контент**
   - Замените информацию на свою
   - Добавьте свои фотографии
   - Отредактируйте описание услуг

## ✏️ Кастомизация

### Изменение основной информации

Отредактируйте в `index.html`:

```html
<!-- Имя и специальность -->
<h1>Доктор [Ваше имя]</h1>
<p>Врач-остеопат, специалист в лечении заболеваний опорно-двигательного аппарата</p>

<!-- Email -->
<a href="mailto:your@email.com">your@email.com</a>

<!-- Телефон -->
<a href="tel:+79991234567">+7 (999) 123-45-67</a>

<!-- Соцсети -->
<a href="https://t.me/yourprofile">Telegram</a>
<a href="https://instagram.com/yourprofile">Instagram</a>
```

### Изменение цветов

Отредактируйте в `styles.css`:

```css
/* Основной цвет (медицинский зеленый) */
background: linear-gradient(135deg, #4CAF50 0%, #45a049 100%);

/* Альтернативные цветовые схемы: */
/* Синий (официальный): #2196F3 - #1976D2 */
/* Красный (срочность): #F44336 - #D32F2F */
/* Фиолетовый (премиум): #9C27B0 - #7B1FA2 */
```

### Замена изображений

```html
<!-- Вместо -->
<img src="https://via.placeholder.com/400x400" alt="Profile">

<!-- Используйте -->
<img src="path/to/your/doctor-photo.jpg" alt="Доктор [Имя]">
```

## 📊 SEO оптимизация

Для улучшения видимости в поисковых системах:

1. **Meta теги:**
   ```html
   <meta name="description" content="Профессиональный остеопат. Лечение заболеваний опорно-двигательного аппарата.">
   <meta name="keywords" content="остеопат, остеопатия, лечение позвоночника">
   ```

2. **Open Graph для соцсетей:**
   ```html
   <meta property="og:title" content="Доктор - Остеопат">
   <meta property="og:description" content="Профессиональные услуги остеопата">
   <meta property="og:image" content="URL_вашего_фото">
   ```

3. **Правильная структура заголовков** (H1, H2, H3)
4. **Оптимизация изображений**
5. **Быстрая загрузка страницы**

## 🚀 Развертывание

### GitHub Pages (бесплатно)
1. Перейдите в Settings репозитория
2. В разделе "Pages" выберите ветку `main`
3. Сохраните
4. Ваш сайт будет доступен по адресу: `https://vsplus07-maker.github.io/Doktor-`

### Другие хостинги
- **Netlify** - Простое развертывание через GitHub
- **Vercel** - Быстрый хостинг с автоматическими деплойментами
- **Heroku** - Полнофункциональный хостинг
- **AWS S3** - Облачное хранилище
